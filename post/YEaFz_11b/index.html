phpMyAdmin提示The mbstring extension is missing. Please check your PHP configuration.的解决方法

2019-02-15 Ubuntu Linux

本文介绍在Linux（本人使用的是Ubuntu）安装phpMyAdmin后，打开http://localhost/phpmyadmin提示“The mbstring extension is missing. Please check your PHP configuration.”的解决方法。

安装phpMyAdmin：

打开终端，输入以下命令行：

sudo apt-get install phpmyadmin

具体步骤不再赘述（安装选项我选的Apache）。

安装完后，phpMyAdmin提示“The mbstring extension is missing. Please check your PHP configuration.”

解决方法：

在你的Apache目录底下，找到php.ini这个文件，使用root权限vi（或vim）编辑它。

以我的目录为例：

sudo vim /etc/php/7.0/apache2/php.ini

在大概888行左右，找到“;extension=php_mbstring.dll”这一条，把前面的分号（即注释）去掉。

然后保存退出。

再重新启动Apache：

service apache2 restart

此时，再打开phpMyAdmin，理论上就没有问题了。
